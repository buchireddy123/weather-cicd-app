# 🌦️ Weather Web App with CI/CD

## 📌 Project Description

This is a Weather Web Application that fetches real-time weather data using an API and displays temperature based on user input.

## 🚀 Features

* Get real-time weather data
* Simple UI
* API Integration (OpenWeather)
* CI/CD using GitHub Actions
* Live deployment using Netlify

## 🛠️ Technologies Used

* HTML, CSS, JavaScript
* GitHub Actions (CI/CD)
* Netlify (Deployment)
